# Next.js Project Setup and Basics

## Overview

This repository contains the setup and basic implementation of a Next.js project using TypeScript and Tailwind CSS. It serves as a foundational boilerplate for building scalable and modular web applications using the Next.js framework.

## Project Structure

The project is structured to facilitate easy navigation and development. The key directories and files include:

```bash
alx-project-0x03/
├── components/
│   ├── common/
│   │   ├── Button.tsx
│   │   ├── Card.tsx
│   │   ├── PostCard.tsx
│   │   ├── PostModal.tsx
│   │   └── UserCard.tsx
│   └── layout/
│       └── Header.tsx
├── interfaces/
│   └── index.ts
├── pages/
│   ├── _app.tsx
│   ├── _document.tsx
│   ├── about.tsx

├── public/
│   └── assets/
│       └── images/

│   └── globals.css
├── .eslintrc.json
├── next.config.js
├── package.json
├── README.md
└── tsconfig.json
```
